🔌 ERP Integration

Already have this business data in SAP or another ERP? Skip the wizard — call these endpoints directly and get back the exact same live, agent-computed analysis this page shows.

Endpoints
GET/api/erp/v1/health
Connectivity + credential check — call once while wiring up the connection.
GET/api/erp/v1/materials
The full 31-material taxonomy with accepted aliases (e.g. "HSD", "PVC resin", "TMT bar") and a SAP material-group hint per item — build your mapping table once from this.
POST/api/erp/v1/impact-analysis
Submit a business profile — materials identified by name or code, logistics, product economics — and get back the same live agentic financial impact analysis shown on this page.
Example request
curl -X POST https://<your-apex-host>/api/erp/v1/impact-analysis \ -H "X-API-Key: <your-erp-api-key>" \ -H "Content-Type: application/json" \ -d '{ "company_name": "Meridian Auto Components", "product": {"name": "Auto parts", "unit_price": 45, "monthly_units": 250000, "pass_through_pct": 35}, "materials": [ {"material": "PVC Resin", "monthly_quantity": 8000, "unit_cost": 90, "currency": "INR", "contract_type": "spot"}, {"material": "HSD", "monthly_quantity": 5000, "unit_cost": 92, "currency": "INR", "contract_type": "fixed", "contract_months_remaining": 4} ], "logistics": {"mode": "road", "monthly_cost": 220000}, "inventory_buffer_days": 18 }'
What comes back
The same fields the results screen on this page shows — narrative, prioritized recommendations, full cost/margin delta, per-material breakdown with live commodity references — plus a material_mapping array confirming exactly how each submitted material name was resolved, so an integration can be audited.
🔑 API key: contact your APEX administrator for a key — it's a shared secret configured server-side (ERP_API_KEY in .env) and is never exposed in this page or sent to your browser. Material names don't need to match APEX's internal codes exactly — "PVC Resin", "polyvinyl chloride", and "PVC" all resolve to the same material.
